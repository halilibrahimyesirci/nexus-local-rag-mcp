# Code of Conduct

## Our pledge

We want taking part in this project to be a welcoming, harassment-free experience for everyone,
regardless of background or identity. Everyone is expected to act with respect and good faith.

## Our standards

- Be respectful, considerate, and constructive.
- Welcome differing viewpoints and accept feedback gracefully.
- Focus on what is best for the project and its community.

Conduct that demeans, harasses, or excludes others is not acceptable.

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its full code of conduct.

## Enforcement

To report a concern, contact the maintainer privately through their GitHub profile
([@halilibrahimyesirci](https://github.com/halilibrahimyesirci)) or via the repository's
[private advisory](https://github.com/halilibrahimyesirci/nexus-local-rag-mcp/security/advisories/new)
form. Reports are reviewed and handled confidentially.
