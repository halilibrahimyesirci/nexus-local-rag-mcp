# Security Policy

## Supported versions

This project is early-stage. Security fixes are applied to the latest `main`.

## Reporting a vulnerability

Please report security issues privately through GitHub's
[private vulnerability reporting](https://github.com/halilibrahimyesirci/nexus-local-rag-mcp/security/advisories/new)
instead of opening a public issue. Reports are acknowledged and addressed as quickly as possible.

The server runs fully offline and stores data locally, so the most relevant areas are parsing
untrusted documents and handling the local SQLite database. Where possible, include reproduction
steps and a sample file.
